Complete MERN project generated at `samagama-faq/` inside the working directory.

**Stack:** Node/Express + MongoDB + React + Tailwind CSS + OpenRouter API (chatbot)

**Key facts:**
- Default admin: admin@samagama.in / Admin@1234
- OpenRouter API key goes in `backend/.env` as `OPENROUTER_API_KEY`
- Start: `docker compose up --build` or manual `npm run dev` in both backend/ and frontend/
- Seed: `cd backend && npm run seed`
- Backend port 5000, Frontend port 5173

**Why:** Built to spec for Samagama internship FAQ with AI chat, Hindi support, Spurti Points FAQs, leave policy, transcript ingestion, ticket escalation, admin panel, dark mode, TTS.
